DragonFable Private Server [WIP]

A private server for DragonFable, allowing custom game and server file locations.

This project allows you to run a private server for DragonFable with customizable game and server file locations.

See SETUP for more information on how to setup the server.

See UPGRADE for more information on how to upgrade the server to a new version with new data and features.

See Development for more information on how to develop the server.
